## Techno utilisée
Next.js

## Installation
```bash
npm install
```
⚠️ Renommer le fichier `template.env.local` en `.env.local` et remplacer/compléter par vos informations

## Utilisation
```bash
npm run dev
```
Un serveur web sera disponible et accessible à l'adresse localhost:3000
