## Techno utilisée
Vue.js

## Installation
```bash
npm install
```
⚠️ Renommer le fichier `template.env` en `.env` et remplacer/compléter par vos informations

## Utilisation
```bash
npm run serve
```
Un serveur web sera disponible et accessible à l'adresse localhost:8080
